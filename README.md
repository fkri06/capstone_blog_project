# Capstone Blog Project

This is a capstone project of section 26 from
[The Complete Full-Stack Web Development Bootcamp.](https://www.udemy.com/course/the-complete-web-development-bootcamp/)

## How to Run the Project

- Download the project
- Change to directory project
```
cd capstone_blog_project
```
- Install dependencies
```
npm install
```
- Run the index.js
```
node indes.js
```

or if you're using `nodemon`

```
nodemon indes.js
```
